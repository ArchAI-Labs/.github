### 🌟 Featured Project: Medha
**Medha** is our current flagship initiative. It is a sophisticated **Semantic Memory system for AI Data Agents**, designed to provide persistent, context-aware memory structures. 

* 🧠 **Mission**: Enabling AI agents to retain long-term information and reason across complex datasets.
* 🚀 **Status**: Under active development (High Priority).
* [**Explore Medha Repository →**](https://github.com/ArchAI-Labs/medha)

### 🛠️ Core Tools
* **[🚀 Code Explainer](https://github.com/ArchAI-Labs/code_explainer)**: Automatically generates clear, structured documentation and UML diagrams directly from your source code.
* **[📊 SonarQube Metrics MCP](https://github.com/ArchAI-Labs/fastmcp-sonarqube-metrics)**: A Model Context Protocol (MCP) server to query SonarQube metrics and technical debt using natural language.
* **[🧠 Cypher Mind](https://github.com/ArchAI-Labs/cypher_mind)**: An AI translator that converts natural language prompts into precise Cypher queries for graph databases.
